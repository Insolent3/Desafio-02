# Trilha HTML - Dio.me
## Desafio 02 - HTML I - Conceitos Básicos

    Site criado para concluir o segundo desafio do curso de HTML da plataforma DIO. 

Este desafio tinha como objetivo, criar um site "quase" completo, com tudo que foi visto no módulo 1 de HTML. Site com navegações entre as páginas, estruturação, mídias, tabelas e um formulário.
